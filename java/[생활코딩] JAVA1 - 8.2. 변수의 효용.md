# [생활코딩] JAVA1 - 8.2. 변수의 효용

```java
public class Latter {

	public static void main(String[] args) {
		String name = "계속";
		System.out.println("안녕하세여."+ name + "..." + name);
		
		
		double VAT = 10.0;
		System.out.println(VAT);
	}

}
>
안녕하세여.계속...계속
10.0
```

- 변수의 이름은 그 변수가 가지는 특성을 고려하고 다른 누군가가 봤을 때 이해할 수 있게 짓는게 중요하다.
