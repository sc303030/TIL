# 중첩 반복문

- 반복문 내부에 또 다른 반복문 존재
- 여러 겹으로 구현 가능

```java
package ch20;

public class NestedLoopTest {

	public static void main(String[] args) {
			
		int dan = 2;
		int count = 1;
		
		for(; dan <= 9; dan++) {
			for(count=1;count <= 9; count++) {
				System.out.println(dan + "X" + count + "=" + dan*count);
			}
			System.out.println();
		}
		
	}

}
```
