# 컴퓨터에서 자료 표현하기

- 컴퓨터에서 자료를 어떻게 표현할까?
  - 기계어로 번역되면 이애함
  - 컴퓨터는 2진수로 이해함, 0과 1로 이해
  - 데이터의 최소 단위를 비트

| 10진수   | 0    | 1             | 2             | 3                             | 4             |
| -------- | ---- | ------------- | ------------- | ----------------------------- | ------------- |
| 2진수    | 0    | 1             | 10            | 11                            | 100           |
| 표현방법 |      | 2<sup>0</sup> | 2<sup>1</sup> | 2<sup>1</sup> + 2<sup>0</sup> | 2<sup>2</sup> |
